## Key Notes

> **Reprocess Video vs Edit Script** — "Reprocess Video" (`POST /reprocess`) re-runs stitch/overlay/mix with cached scenes, skipping expensive Veo/VO calls. "Edit Script" (`PATCH /spec`) clears doc_spec and re-runs everything from scratch. Use Reprocess to pick up ffmpeg bug fixes; use Edit Script to change content.

> **Veo 3.1 pricing** — $0.35/second is an estimate. Actual rate may differ. Cost records now written automatically to `api_costs` table — query `GET /projects/{id}/cost-breakdown` to verify.

> **Clerk dev keys in production** — Console shows `pk_test_` keys. Dev keys have rate limits. Acceptable for low-volume internal testing; swap for `pk_live_` before public launch.

> **Tommy's video** — First successful 30s video generated 2026-03-20. Use "Reprocess Video" button on his project page to recomposite with latest ffmpeg quality fixes (overlay + tail silence).
